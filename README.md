# Albacheck SNS Statistics
각 SNS 의 API 를 연동해서 Albacheck 의 SNS 영향도를 분석합니다.

- Server URL
    - https://albacheck-sns-statistics.herokuapp.com
## Youtube

### 통계자료 가져오기

- API
    - GET /v1/statistics/youtube
- Request
- Response
```json
{
    "commentCount": 0,
    "hiddenSubscriberCount": true,

    "videoCount": 18,
    "viewCount": 637
}
```
